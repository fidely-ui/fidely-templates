# Vite & Fidely UI Preset

A minimal starter for bringing Fidely’s Panda CSS preset into your project.
styles only, no components. Build and style your own UI freely.

## What this template gives you

- Fidely design tokens & recipes via presets

- Panda CSS powered styling system

- Full control over your own components

- Zero UI lock-in

## Technologies Used

- Vite
- Fidely UI Presets
- Panda CSS
- TypeScript
- next-themes

### Getting Started

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/panda-preset-starter
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
