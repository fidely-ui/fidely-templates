# Vite & Fidely UI Template

A starter template for building modern web applications with **Vite** and **Fidely UI v2** 

For Fidely UI v1 check v1 branch

## Technologies Used

- Vite
- Fidely UI - v2
- Panda CSS
- JavaScript
- next-themes

### Getting Started

run the following command:

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/vite-js-template
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
