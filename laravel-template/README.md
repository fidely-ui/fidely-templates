# Laravel & Fidely UI Template

A starter template with **Laravel 13** and **Fidely UI v2** 

## Technologies Used

- Laravel 13
- PHP v8.3
- Fidely UI v2
- Panda CSS
- TypeScript
- next-themes

### Getting Started

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/laravel-template
```

### Install dependencies

```bash
composer install & npm install
```

### Run the development server

Start the frontend development server:

```bash
npm run dev
```

Start the backend development server:

```bash
php artisan server 
```

if using Herd


```bash
herd php artisan server 
```
