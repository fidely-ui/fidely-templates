# Next.js & Fidely UI Template

A starter template for building modern web applications with **Next.js (App Router)** and **Fidely UI v2** 

For Fidely UI v1 check v1 branch

## Technologies Used

- Next.js 16 (App Router)
- Fidely UI - v2
- Panda CSS
- TypeScript
- next-themes

## Getting Started

### Create a new project

run the following command:

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/next-app-template
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
