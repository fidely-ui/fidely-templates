# Create React Router & Fidely UI Template

A starter template for building modern web applications with **Vite**, **React Router** and **Fidely UI v2** 

## Technologies Used

- Vite
- React Router (v7)
- Fidely UI (v2)
- Panda CSS
- TypeScript
- next-themes

### Getting Started

run the following command:

```bash
npx gitpick https://github.com/fidely-ui/fidely-templates/tree/main/react-router-template
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
